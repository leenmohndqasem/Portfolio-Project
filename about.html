About Me

I am a motivated and detail-oriented front-end developer with a strong interest in creating clean, responsive, and user-friendly websites. I enjoy turning ideas into real projects using HTML, CSS, and JavaScript, while focusing on writing organized and readable code.

I believe that good design is not only about how a website looks, but also about how it works and how users interact with it. For this reason, I always try to build layouts that are simple, clear, and easy to navigate on all screen sizes.

I am constantly learning new technologies and improving my skills through practice and real projects. My goal is to grow as a developer, gain more experience, and create meaningful digital solutions that provide value and a pleasant user experience.

Skills

HTML – CSS – JavaScript – React

Front-end development focuses on building the user interface and visual part of the website. Using HTML and CSS to structure and style pages, JavaScript to add interactivity, and React to create dynamic and reusable components.

Node.js – Python – REST APIs – Authentication

Back-end development handles server-side logic and application functionality. Node.js and Python are used to process requests, manage users, and build APIs that connect the front-end with the database securely and efficiently.

SQL – MongoDB – Git – GitHub

Databases are used to store and manage application data using SQL or MongoDB. Git and GitHub are essential tools for version control, tracking changes, and collaborating on projects in a professional development workflow.
